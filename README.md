# Intego Landing Page

## How to run project:

### 1. Run server
- cd ./server/
- run: `npx nodemon@latest index.js`

### 2. Run in addition the front end
- cd ./frontend
- run: `npm run dev`

## PHP price converter:
You can find this file - server/CurrencyConverter.php

## Database file:
- MySql was used
- You can find the exported DB file in the main folder
